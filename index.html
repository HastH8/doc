Function: check_password_strength

Function: validate_phone_number

Validates a phone number to ensure it meets specific criteria:

  • Ensures the phone number is a valid 10-digit number.
  • Checks if the area code belongs to a predefined list of Ontario area codes.
    • 416
    • 647
    • 437
    • 519
    • 226
    • 548
    • 613
    • 343
    • 705
    • 249
    • 807
    • 905
    • 289
    • 365
  • Formats the phone number to a standard format if valid.

Algorithms

1. Password Strength Checker Algorithm

  1. Start with a score of 0
  2. Check password length (8-15 characters)
    • If valid: Add 1 point
    • If invalid: Add feedback message
  3. Check for spaces
    • If no spaces: Add 1 point
    • If spaces found: Add feedback message
  4. Check for digits
    • If contains digit: Add 1 point
    • If no digit: Add feedback message
  5. Check for uppercase letters
    • If contains uppercase: Add 1 point
    • If no uppercase: Add feedback message
  6. Check for special characters
    • If contains special character: Add 1 point
    • If no special character: Add feedback message
  7. Additional checks (subtract 1 point each if found):
    • Check against common password list
    • Check for sequential characters
    • Check for repeated characters
  8. Determine final strength based on score:
    • 5+ points: Very Strong
    • 4 points: Strong
    • 3 points: Moderate
    • 2 points: Weak
    • Below 2: Very Weak

2. Phone Number Validator Algorithm

  1. Remove all non-digit characters from input
  2. Check if cleaned number is exactly 10 digits
    • If not 10 digits: Return invalid
  3. Extract first 3 digits as area code
  4. Check if area code is in Ontario area codes list
    • If not in list: Return invalid
  5. Format valid number as: (XXX) XXX-XXXX
  6. Return success with formatted number

Route: index

Processes form submissions for password strength validation and phone number validation. Handles POST requests with the following logic:

  • If a password is submitted, it checks its strength and provides feedback.
  • If a phone number is submitted, it validates the phone number and provides a message.
